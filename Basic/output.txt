To print any output on screen we use print() function. 
It is a built-in function in Python. 
It is used to print the output on the screen.

Syntax of print() function:
print(*objects, sep=' ', end=' ', file=sys.stdout, flush=False)

Parameters of print() function:
1. objects: It is a variable-length argument list.
2. sep: It is used as separator between the values. It defaults into a space character.
3. end: It is used to append at the end of the output. It defaults into a new line.
4. file: It is an object with a write method. It defaults into the sys.stdout.
5. flush: It is a Boolean, controlling whether to forcibly flush the stream. It defaults into False.

Return value of print() function:
It returns None.

Cases:
1. integer: %d
2. float: %f
3. string: %s
4. character: %c
5. octal: %o
6. hexadecimal: %x
7. exponential: %e
